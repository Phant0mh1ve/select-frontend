# Тестовое Angular приложение

Небольшое приложение на Angular, реализующее работу со списками элементов и переключением разделов.

## Демо

Приложение доступно по ссылке:

https://YOUR_USERNAME.github.io/select-frontend/

## Функциональность

- переключение между разделами **Тип 1** и **Тип 2**
- отображение списка элементов
- выбор элементов с помощью чекбоксов
- подсчёт количества выбранных элементов
- подсчёт общей суммы значений
- сброс выбранных элементов при переключении раздела
- отображение изображения для каждого раздела
- базовая адаптивная верстка

## Используемые технологии

- **Angular**
- **TypeScript**
- **HTML**
- **CSS**

## Установка и запуск

Клонировать репозиторий:

git clone https://github.com/Phant0mh1ve/select-frontend.git

Перейти в папку проекта:

cd select-frontend

Установить зависимости:

npm install

Запустить приложение:

ng serve

Открыть в браузере:

http://localhost:4200

## Сборка проекта

Для сборки проекта выполнить:

ng build

## Деплой на GitHub Pages

Сборка проекта:

ng build --base-href="/select-frontend/"

Публикация:

npx angular-cli-ghpages --dir=dist/select-frontend/browser
